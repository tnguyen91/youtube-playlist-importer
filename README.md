# YouTube Playlist Importer

A simple React + Tailwind web app that lets you import a public YouTube playlist, view its videos, and save them locally in your browser.

---

## Features

- Paste a YouTube playlist URL and view its contents
- See thumbnails, titles, and channels for each video
- Save named playlists to localStorage
- Load saved playlists

---

## Built With

- React (with Vite)
- TailwindCSS
- YouTube Data API v3

---

## Getting Started

### Clone and run locally:

```bash
git clone https://github.com/YOUR_USERNAME/youtube-playlist-importer.git
cd youtube-playlist-importer
npm install
npm run dev
